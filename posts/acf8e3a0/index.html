基于 DWR + ActiveMQ 的 Web 端订阅消息推送

发表于 2017-05-25 | 分类于 Java Web | 阅读次数：

基于 Web 的消息订阅／发布系统可以分两部分考虑：一是使用什么工具搭建订阅／发布系统，二是如何进行 Web 前后端交互。

对于前者，业界很多 MQ 产品可以满足需求：RabbitMQ、ActiveMQ、ZeroMQ 等，本项目采用 ActiveMQ。关于 Web 前后端交互，目前最流行的技术是 HTML5 的 WebSocket，然而项目方要求兼容 IE8，因此只能寻求旧一点的技术，DWR 完全能满足项目的需求。

接下来介绍如何把这两种技术整合进基于 Spring + SpringMVC 的 Java Web 后台。
